Creating a JSON file for parsing. Starting with the sample code: 

{  
"Countries": [  
{ "Name": "India",  
"Capital": "Delhi"  
},  
{  
"Name": "France",  
"Major": "Paris"  
},  
]  
} 


The information above is for two countries. The rules are as follows.
1. { } are containers
2. [ ] represent arrays/lists
3.  :  Seperate Names and values 
4.  ,  Seperate array elements 
5. " " Hold text strings
